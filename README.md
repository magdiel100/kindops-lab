# kindops-lab

Laboratorio pratico para aprender DevOps/SRE com foco em implementacao real, documentacao e evolucao incremental.

## Pre-requisitos
- Git
- Docker
- kubectl
- kind
- Helm
- Terraform
- Ansible
- Make

## Estrutura do projeto
- `apps/`
- `infra/terraform/`
- `infra/ansible/`
- `charts/`
- `gitops/`
- `observability/`
- `docs/`

## Inicio rapido
```bash
make bootstrap
```
