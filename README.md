# kindops-lab

Laboratório prático para aprender DevOps/SRE com foco em implementação real, documentação e evolução incremental.

## Pre-requisitos
- Git
- Docker
- kubectl
- kind
- Helm
- Terraform
- Ansible
- Make

## Estrutura do projeto
- `apps/`
- `infra/terraform/`
- `infra/ansible/`
- `charts/`
- `gitops/`
- `observability/`
- `docs/`

## Início rápido
```bash
make bootstrap
```
